### 연구 : 만화도 Json으로 분석하고 변경이 가능할까?

### 1. 원본 이미지가 필요하다. 마음에 드는 이미지 웹툰 만화 또는 내가 그린 그림

### 2. 그 이미지를 분석하고 제이슨으로 변경할 프롬프트가 필요하다.

```jsx
Analyze this webtoon/manga panel and convert all visual and narrative information into a highly detailed, structured JSON format. Focus specifically on isolating artistic elements, character features, and composition. For each key element, extract its precise **art style** (e.g., cel-shading, painterly, line-heavy), **color palette**, and **compositional role**. Include JSON keys for:

- `art_style`: The overall rendering method and line work style.
- `atmosphere_and_mood`: The emotional tone of the scene (e.g., tense, romantic, melancholic).
- `overall_color_palette`: A list of dominant colors or hex codes used in the panel.
- `characters`: An array containing `name` (if identifiable or generic description), `expression`, `clothing_details`, and `pose`.
- `visual_elements`: An array containing `background_setting`, `lighting_source`, and `special_effects` (e.g., speed lines, sparkles, screentones).
- `composition`: Key for `camera_angle` (e.g., high angle, close-up) and `panel_layout`.

Output **ONLY** valid JSON and format the output as a copyable JSON code block using Markdown.
```
