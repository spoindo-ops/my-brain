안녕하세요 우리 대표님들,

어제 오늘 개발한 AI 비서 테스트 파일 드립니다. 함께 연구해봐요. 

본 가이드는 스마트폰의 **텔레그램**을 통해 회사의 AI 비서와 자연어로 대화하고, 비서가 직접 **구글 캘린더** 일정을 관리하도록 만드는 셋업 과정을 다룹니다. (소요 시간: 약 10분)

---

## 🟢 Module 1. 텔레그램 봇 API 연결 (5분)

AI 비서의 '입'과 '귀'가 되어줄 텔레그램 메신저를 연결하는 단계입니다.

### Step 1. 봇 생성 및 토큰 발급

1. 텔레그램 앱에서 [@BotFather](https://t.me/BotFather)를 검색하여 대화를 시작합니다.
2. `/newbot` 명령어를 입력합니다.
3. **봇의 이름**을 정합니다. (예: `나의 스마트 비서`)
4. **봇의 Username**을 정합니다. (반드시 `bot`으로 끝나야 함 / 예: `my_ai_sec_bot`)
5. 발급된 **API Token**(`123456789:AAEx...`)을 안전한 곳에 메모합니다.

### Step 2. 대화 세션 활성화

1. 생성된 봇의 링크를 클릭하여 대화창에 진입합니다.
2. **[시작(Start)]** 버튼을 누릅니다.
    
    > **Note:** 메시지를 최소 한 번 이상 보내야 시스템에서 사용자의 `chat_id`를 인식할 수 있습니다.
    > 

### Step 3. IDE(VS Code/Cursor) 연동

1. 사이드바의 **Connect AI** 아이콘 → **비서(Secretary)** 메뉴를 선택합니다.
2. **"📨 텔레그램 연결 필요"** 카드를 클릭하거나 설정(⚙️)으로 이동합니다.
3. 다음 과정을 진행합니다:
    - **봇 토큰 입력**: 발급받은 API 토큰을 붙여넣습니다.
    - **Chat ID 선택**: 자동 감지된 본인의 계정을 클릭합니다.
    - **연결 테스트**: 테스트 메시지가 핸드폰으로 도착하면 성공입니다. ✅

---

## 🔵 Module 2. 구글 캘린더 권한 연동 (5분)

비서가 일정을 읽고, 쓰고, 수정할 수 있도록 권한을 부여하는 단계입니다.

### Step 1. OAuth 인증 프로세스 시작

1. IDE에서 `Cmd+Shift+P` (Mac) 또는 `Ctrl+Shift+P` (Windows)를 누릅니다.
2. "Connect AI: Google Calendar 자동 일정 연결 📅"을 검색하여 실행합니다.

### Step 2. 권한 허용

1. 자동으로 브라우저가 실행됩니다. 연결할 **구글 계정**을 선택합니다.
2. "Connect AI가 캘린더에 접근하려고 합니다"라는 메시지가 나오면 [허용]을 클릭합니다.
3. "인증 완료" 페이지가 뜨면 브라우저를 닫고 IDE로 복귀합니다.

### Step 3. 최종 작동 확인

- 텔레그램 대화창에 "오늘 일정 알려줘"라고 입력합니다.
- 비서가 캘린더 데이터를 읽어와 답변한다면 모든 설정이 완료된 것입니다.

## 🆘 문제 해결 (Troubleshooting)

**Q1. 텔레그램 비서가 대답을 안 해요.**

- 봇 토큰이 정확한지 확인하세요. (`BotFather` > `My Bots`에서 재확인 가능)
- 봇 대화방에서 `/start`를 눌렀는지 확인하세요.

**Q2. 똑같은 대답이 두 번씩 옵니다.**

- VS Code나 Cursor 창이 여러 개 열려 있는지 확인하세요. 활성화된 창의 개수만큼 비서가 작동합니다.

**Q3. 캘린더 연동이 갑자기 안 돼요.**

- OAuth 보안 토큰이 만료되었을 수 있습니다. [Module 2]의 과정을 다시 수행하여 인증을 갱신하세요.
