## 최종 사운드 디자인 및 시각 가이드라인 일관성 검증 체크리스트

CEO 지시에 따라, 확정된 음악 스타일과 'Deep Navy/Gold' 비주얼 시스템 간의 청각적 경험 일관성을 100% 검증하기 위한 최종 체크리스트를 작성합니다. 이 체크리스트는 모든 에셋 제작 단계에서 시각(Visual)과 청각(Auditory)이 하나의 통일된 감정적 흐름을 전달하는지 확인하는 데 사용됩니다.

---

### I. 핵심 비주얼 시스템 (Deep Navy/Gold) 검증 항목

| No. | 항목 (Visual System Check) | 기준 (Standard) | 적용 여부 (Status) | 코멘트 (Notes) |
| :---: | :--- | :--- | :--- | :--- |
| **1** | **색상 팔레트 일관성** | Deep Navy (#000080 계열), Gold (#FFD700 계열)의 명암비 및 채도 비율이 모든 에셋(썸네일, 인트로, 배너)에서 동일하게 적용되었는가? | [ ] | |
| **2** | **타이포그래피 강조** | 핵심 키워드('Source. Trust. Result.') 및 브랜드 로고/인증 마크의 Gold 색상 사용 위치와 크기가 시각적 계층 구조를 명확히 하는가? | [ ] | |
| **3** | **질감 및 분위기 (Texture)** | 영상 배경(Navy)과 강조 요소(Gold) 간의 대비(Contrast)가 프리미엄하고 신뢰감을 주도록 설정되었는가? (과도한 자극 방지) | [ ] | |
| **4** | **QA 체크리스트 준수** | 사전에 정의된 Visual Sign-off QA 체크리스트(신뢰 증명 요소 포함)가 모든 최종 에셋에 100% 반영되었는가? | [ ] | |

### II. 청각적 경험 (Sound Design) 검증 항목

| No. | 항목 (Auditory Experience Check) | 기준 (Standard) | 적용 여부 (Status) | 코멘트 (Notes) |
| :---: | :--- | :--- | :--- | :--- |
| **1** | **선택된 음악 스타일 일치** | 최종 선택된 음악 장르 및 분위기(예: Cinematic, Ambient, Epic 등)가 'Deep Navy/Gold'의 프리미엄 신뢰감과 완벽하게 조화되는가? | [ ] | (선택된 스타일 명시 필요) |
| **2** | **청각적 흐름 제어** | 영상의 주요 전환점(Hook, Value Point, CTA)에서 음악의 템포, 볼륨, 악기 구성이 시청자의 감정적 흐름을 의도대로 완벽히 제어하는가? | [ ] | (Trust Capital 증명 목표 반영) |
| **3** | **사운드-비주얼 동기화** | 시각적인 클라이맥스(Gold 강조)와 청각적인 절정(음악의 고조)이 정확히 1:1로 일치하여 몰입감을 극대화하는가? | [ ] | |
| **4** | **앰비언스 및 신뢰감** | 배경 사운드(Ambient Layer)가 지나치게 산만하지 않고, '신뢰 자본(Trust Capital)'을 구축하는 안정적이고 권위 있는 분위기를 유지하는가? | [ ] | |

### III. 최종 통합 검증 (Final Synthesis)

| :---: | :--- | :--- | :--- |

| **2** | 모든 에셋에서 'Source. Trust. Result.' 메시지가 시각과 소리로 일관되게 전달되는가? | [ ] | |
| **3** | 최종 결과물은 프리미엄 브랜드의 기대치(Visual Sign-off)를 100% 충족하는가? | [ ] | |
