### 1. 유튜브 썸네일 디자인 가이드라인 (Video Thumbnail)

**목표:** 신뢰 자본(Trust Capital)과 품질 증명(Quality Proof)을 극대화하여 높은 클릭률(CTR) 유도.
**적용 컨셉:** 이전 단계에서 가장 높은 CTR 예측치를 보인 **컨셉 B**를 기반으로, 권위적인 Deep Navy/Gold 톤앤매너 적용.

| 항목 | 상세 사양 (Visual Sign-off 기준) | 적용 코드 및 설명 |
| :--- | :--- | :--- |
| **배경 색상** | Deep Navy (권위와 깊이 강조) | `#0A1931` (Deep Navy) |
| **강조 색상** | Gold (프리미엄, 신뢰 자본 증명) | `#FFD700` (Gold) |
| **주요 텍스트 (후크)** | 핵심 메시지 (예: "신뢰 자본 증명") | Gold (#FFD700), 세리프체(Serif) |
| **보조 텍스트** | 부제 또는 보조 설명 | White (`#FFFFFF`), 세리프체(Serif) |
| **레이아웃 좌표** | 중앙 좌측 배치 (안전 영역 고려) | 핵심 메시지: (X: 10% ~ 60%), 보조 텍스트: (Y: 50% ~ 80%) |
| **디자인 요소** | 금색 인증 마크/심볼 | Gold (`#FFD700`)로 처리하여 중앙에 배치. (Visual Sign-off 필수) |
| **폰트 스타일** | 제목: Playfair Display Bold 또는 Baskerville Semi-Bold (세리프체) | 제목 텍스트는 크기와 두께를 명확히 구분하여 권위를 부여함. |
| **QA 체크리스트 반영 사항** | 1. 색상 대비(Navy/Gold)가 충분한가? (Pass) / 2. 인증 마크의 위치와 크기가 일관적인가? (Pass) / 3. 메시지가 명확하게 전달되는가? (Pass) | 모든 요소는 Deep Navy 배경 위에서 Gold로 명확히 분리되어야 함. |

**🎨 최종 디자인 요약:**
Deep Navy 배경 위에 중앙에 골드 색상의 '품질 인증 마크'를 배치하고, 그 주변에 흰색 또는 골드 색상으로 "신뢰 자본 증명"과 같은 핵심 메시지를 세리프체로 강조하여 시각적 무게중심을 확보합니다.

---

### 2. 채널 배너 아트 디자인 가이드라인 (Channel Banner Art)

**목표:** 채널의 전체적인 권위와 전문성을 일관되게 전달하며 구독 유도.
**적용 컨셉:** 비주얼 시스템(Deep Navy/Gold)을 채널 영역 전체에 확장 적용하여 통일성 확보.

| 항목 | 상세 사양 (Visual Sign-off 기준) | 적용 코드 및 설명 |
| :--- | :--- | :--- |
| **배경 색상** | Deep Navy (전체 영역) | `#0A1931` |
| **상단/중앙 강조** | 채널 이름 또는 핵심 슬로건 | Gold (`#FFD700`) |
| **구조** | 좌측: 브랜드 로고/이름, 중앙: 핵심 슬로건, 우측: 비주얼 시스템의 미묘한 패턴 또는 인증 요소 배치. | 전체적으로 정적인 느낌을 주어 신뢰감을 극대화함. |
| **타이포그래피** | 채널 이름 폰트 (Playfair Display 등)는 Gold 색상으로, 슬로건은 White로 처리하여 계층 구조를 명확히 함. | 모든 텍스트는 세리프체(Serif)를 유지해야 함. |
| **QA 체크리스트 반영 사항** | 1. Deep Navy와 Gold의 비율이 채널 전체에 균일하게 적용되었는가? (Pass) / 2. 시각적 요소들이 영역 내에서 조화롭게 배치되었는가? (Pass) | 배너 영역 내 모든 디자인은 썸네일과 동일한 색상 및 톤앤매너를 유지해야 함. |

**🎨 최종 디자인 요약:**
채널 전체를 Deep Navy로 채우고, 상단 또는 중앙에 Gold 색상으로 채널 이름/핵심 가치를 배치하여 시각적 위계를 설정합니다. 모든 요소는 정제되고 권위적인 느낌을 주도록 최소한의 디자인만 사용합니다.
