# 💎 Deep Navy/Gold 시각 시스템 정의 파일 (개발팀 전달용)

이 문서는 'Trust Capital'을 시각적으로 증명하기 위한 핵심 브랜드 시스템의 모든 규칙과 코드를 담고 있습니다. 개발팀(코다리)은 이 정의를 기반으로 자동화 모듈(`Visual Validator Module`)을 구현해야 합니다.

## 1. 색상 팔레트 (Color Palette)
브랜드의 신뢰감과 프리미엄을 표현하는 핵심 색상입니다. 모든 디자인 에셋에 일관되게 적용되어야 합니다.

| 용도 | 색상 이름 | HEX 코드 | RGB 값 | 사용 목적 |
| :--- | :--- | :--- | :--- | :--- |
| **Primary Background** | Deep Navy | `#0A1931` | (10, 25, 49) | 메인 배경, 신뢰감 표현 |
| **Accent Gold** | Rich Gold | `#FFD700` | (255, 215, 0) | 강조, 인증 마크, 프리미엄 요소 |
| **Text/Primary** | Ivory White | `#F8F8F8` | (248, 248, 248) | 본문 텍스트 |
| **Secondary Accent** | Soft Gold | `#B8860B` | (184, 134, 11) | 보조 강조 요소 |

## 2. 타이포그래피 시스템 (Typography System)
가독성과 브랜드의 고급스러움을 동시에 확보하기 위한 서체 규칙입니다.

| 레벨 | 역할 | 폰트 패밀리 | 굵기 (Weight) | 크기 (Size, px) | 사용 예시 |
| :--- | :--- | :--- | :--- | :--- | :--- |
| **H1** | 메인 헤드라인 | Montserrat | Bold (700) | 36px | 영상 제목, 핵심 메시지 |
| **H2** | 섹션 제목 | Montserrat | SemiBold (600) | 24px | 콘텐츠 섹션 구분 |
| **Body** | 본문 텍스트 | Roboto | Regular (400) | 16px | 상세 설명, 본문 내용 |
| **Caption** | 보조 정보 | Roboto | Light (300) | 12px | 저작권, 작은 참고 사항 |

## 3. 비주얼 시스템 규칙 (Visual System Rules)
모든 시각 자료에 적용될 핵심 레이아웃 및 강조 규칙입니다.

*   **배경:** 모든 주요 콘텐츠 영역은 **Deep Navy (`#0A1931`)**를 기본 배경으로 사용합니다.
*   **강조:** 핵심 정보, 인증 마크, CTA 버튼 등은 반드시 **Rich Gold (`#FFD700`)**로 처리하여 시각적 계층 구조를 만듭니다.
*   **여백 (Padding):** 콘텐츠 내 요소 간의 충분한 여백을 확보하여 고급스러운 느낌을 유지합니다. 최소 패딩은 24px 이상을 권장합니다.
*   **인증 마크:** 모든 최종 결과물에는 **Rich Gold** 색상으로 처리된 독자적인 '품질 인증' 마크를 삽입해야 합니다.

## 4. Visual Sign-off QA 체크리스트 (QA Checklist)
Editor 및 자동화 시스템이 시각적 일관성을 보장하기 위해 반드시 확인해야 할 항목입니다.

| No. | 검토 항목 | 기준 (Deep Navy/Gold 적용 여부) | 상태 (Pass/Fail) | 비고 |
| :--- | :--- | :--- | :--- | :--- |
| 1 | 배경 색상 일치 | 전체 배경이 `#0A1931`인가? | | |
| 2 | 강조 색상 사용 | 핵심 요소가 `#FFD700`으로 강조되었는가? | | |
| 3 | 타이포그래피 규칙 준수 | H1/H2/Body의 폰트 및 크기가 정의된 규칙을 따르는가? | | |
| 4 | 품질 인증 마크 | '품질 인증' 마크가 Gold 색상으로 명확히 표시되었는가? | | |
| 5 | 여백 준수 | 요소 간 패딩이 최소 24px 이상인가? | | |
