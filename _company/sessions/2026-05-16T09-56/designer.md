## 🖼️ 시각 시스템 최종 확정: Deep Navy/Gold (신뢰 기반 포지셔닝)

**핵심 원칙:** 권위(Authority), 프리미엄(Premium), 신뢰(Trust)를 극대화하며, 모든 요소는 금색(Gold)으로 강조되어야 합니다.

| 요소 | 설정 값 | 상세 설명 및 적용 방안 |
| :--- | :--- | :--- |
| **주요 배경색 (Primary BG)** | `#0A1931` (Deep Navy) | 깊고 어두운 남색 계열로 신뢰와 고급스러움을 표현. 영상의 심각성과 품질을 강조. |
| **강조 색상 (Accent Color)** | `#FFD700` (Gold) | 금색으로, 'Source', 'Trust', 'Result' 키워드 및 핵심 인증 마크에만 사용. 시선을 즉각적으로 집중시킴. |
| **보조 색상 (Secondary BG)** | `#1E2A43` (Slightly Lighter Navy) | 텍스트 영역이나 구분선에 사용하여 깊이감을 더함. |
| **주요 타이포 (Heading Font)** | 'Playfair Display Bold' 또는 유사 세리프체 | 권위 있고 클래식한 느낌을 주어 프리미엄 이미지를 구축. |
| **보조 타이포 (Body Font)** | 'Montserrat Regular' 또는 'Inter' | 가독성을 최우선으로 하여 정보 전달의 명확성을 확보. |

## 🎬 채널 아트 및 배너 에셋 디자인 가이드라인 초안

### 1. 채널 아트 (Channel Art / Banner) - 최종 지침

**목표:** 채널 전체에 'Deep Navy/Gold' 시스템을 즉각적으로 각인시키고, 콘텐츠의 포지셔닝(품질 인증)을 시각화하여 신뢰도를 극대화합니다.

| 항목 | 디자인 상세 지침 (Layout & Style) | 적용 요소 |
| :--- | :--- | :--- |
| **레이아웃** | 가로형 비율 (16:9 또는 2560x1440 권장). 중앙 집중형으로 핵심 메시지를 배치. | - |
| **배경** | 전체 배경은 **Deep Navy (`#0A1931`)** 단색으로 처리. | 배경 색상 |
| **핵심 시각화** | 좌측 또는 우측에 Gold 라인이나 마스크팩/재료의 클로즈업 이미지를 배치하여 'Source'를 암시. | 원료 이미지, 금색 강조선 |
| **메시지 배치 (중앙)** | 중앙 상단에 핵심 가치(Source. Trust. Result.)를 Gold 텍스트로 배치하고, 그 아래에 영상 주제를 명확히 제시. | `Source. Trust. Result.` (Gold), 영상 제목 (White/Gold) |
| **브랜딩 요소** | 채널 로고 또는 'Quality Certified'와 같은 인증 마크는 우측 하단에 작게 Gold로 배치하여 신뢰도를 보강. | 인증 마크 (Gold) |
| **톤앤매너** | 어둡고, 차분하며, 금색의 대비를 통해 고급스러움을 극대화. 정보 전달보다는 '품질 증명'이라는 인상을 주어야 함. | 분위기 설정 |

### 2. 배너 에셋 (Banner Assets) - 확장 적용 지침

배너는 채널 아트보다 더 많은 정보를 담을 수 있으므로, **세분화된 섹션 분리**를 통해 정보의 계층 구조(Hierarchy)를 명확히 해야 합니다.

| 섹션 | 내용 및 디자인 지침 | 시각적 강조 (Color/Font) |
| :--- | :--- | :--- |
| **A. 핵심 가치 (Top Bar)** | `Source. Trust. Result.` 세 단어를 가장 크게 배치하여 브랜드의 철학을 명시. | Gold 텍스트, 큰 세리프체 |
| **B. 비주얼 예시 (Middle Section)** | 실제 제품 또는 제조 공정의 고화질 이미지를 배치하여 'Source'에 대한 시각적 증거 제시. | 이미지 중심, 배경은 Deep Navy |
| **C. 콜투액션 (Bottom Bar)** | 채널 구독 유도 문구 ("신뢰 기반 화장품 제조의 모든 것") 및 CTA 버튼 영역 확보. | White 텍스트 위에 Gold 프레임으로 강조 |
