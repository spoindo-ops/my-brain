# 💎 Deep Navy/Gold Visual System: 디자인 브리프 (초안)

## 1. 브랜드 비주얼 시스템 정의 (Visual System Definition)

**핵심 철학:** 'Source. Trust. Result.' (출처. 신뢰. 결과)를 시각적으로 증명하며, 무역 및 화장품 재료 분야의 **신뢰성과 프리미엄 가치**를 극대화한다.

**주요 색상 팔레트 (Color Palette)**
| 용도 | 색상 이름 | HEX 코드 | RGB 값 | 역할 및 의미 |
| :--- | :--- | :--- | :--- | :--- |
| **Primary (배경)** | Deep Navy | `#0A1931` | (10, 25, 49) | 깊이, 신뢰, 전문성, 밤하늘의 고급스러움. 모든 콘텐츠의 주 배경색으로 사용. |
| **Accent (강조)** | Gold Accent | `#FFD700` | (255, 215, 0) | 품질 인증, 프리미엄 가치, 성공, 희소성 강조. 핵심 CTA 및 인증 마크에 사용. |
| **Secondary (텍스트/보조)** | Soft Ivory | `#F5F5F5` | (245, 245, 245) | 배경색 대비를 위한 부드러운 밝은 톤. 본문 텍스트 영역에 사용. |
| **Neutral (보조)** | Deep Slate | `#1E273B` | (30, 39, 59) | 그림자 및 보조 요소의 깊이를 더하는 데 사용되는 어두운 회색 계열. |

**타이포그래피 규칙 (Typography Rules)**
*   **헤드라인 (H1/Title):** **Playfair Display Bold** 또는 **Cinzel Decorative**. (고급스럽고 권위 있는 느낌 강조)
*   **본문 및 설명 (Body/Caption):** **Noto Sans KR Medium**. (가독성과 전문성을 확보하여 신뢰도를 높임)
*   **강조 텍스트 (Accent):** 헤드라인에 Gold Accent 색상을 적용하거나, 핵심 수치(QA 결과 등)에만 Gold를 사용하여 시각적 계층 구조를 명확히 한다.

## 2. 레이아웃 및 구성 원칙 (Layout & Composition Principles)

**A. 유튜브 썸네일 가이드라인 (Thumbnail Guideline)**
*   **배경:** **Deep Navy (`#0A1931`)**을 기본 배경으로 설정하여 깊이감을 부여한다.
*   **강조 요소:** 핵심 메시지나 숫자(QA 결과, 프리미엄 강조 문구)는 **Gold Accent (`#FFD700`)**로 처리하여 시선이 즉시 집중되도록 한다.
*   **레이아웃:**
    1.  좌측 또는 상단에 제품/결과 이미지 배치 (신뢰도 확보).
    2.  우측 또는 중앙에 핵심 문구(후크)를 Gold 텍스트로 배치.
    3.  '품질 인증 마크' 영역을 반드시 명확히 분리하여 Gold 색상으로 강조한다.

**B. 인스타그램 피드 가이드라인 (Instagram Feed Guideline)**
*   **일관성:** 모든 포스트는 **Deep Navy 배경의 톤앤매너**를 유지하며, 제품이나 재료 이미지는 고해상도와 명암 대비가 높은(High Contrast) 스타일을 지향한다.
*   **텍스트 강조:** 본문 설명은 Soft Ivory(`F5F5F5`) 또는 Deep Slate(`1E273B`) 위에서 읽기 쉽게 배치하며, 핵심 가치 문구에는 Gold 계열의 포인트를 적용하여 시각적 프리미엄을 부여한다.
*   **QA 증명:** '품질 보증 투자'와 같은 메시지는 반드시 **Gold Accent** 톤으로 처리하여 신뢰 자본을 즉각적으로 전달해야 한다.

이 체크리스트는 모든 시각 에셋이 디자인 가이드라인을 준수했는지 확인하는 최종 검토 기준입니다.

| 항목 | 기준 | 적용 여부 (Yes/No) | 비고 |
| :--- | :--- | :--- | :--- |
| **색상 일관성** | Deep Navy와 Gold Accent의 비율이 70:30 이상 유지되는가? | | |
| **타이포그래피 적용** | 헤드라인에 Playfair/Cinzel, 본문에 Noto Sans KR만 사용했는가? | | |
| **신뢰 요소 반영** | 모든 디자인에 '품질 인증'을 상징하는 Gold 마크 또는 시각적 요소를 포함했는가? | | |
| **명암 대비** | 어두운 배경(Navy) 위에서 강조색(Gold)의 대비가 충분하여 가독성이 확보되었는가? | | |
| **메시지 일관성** | 콘텐츠의 핵심 메시지('Source. Trust. Result.')가 시각적으로 명확하게 전달되는가? | | |
