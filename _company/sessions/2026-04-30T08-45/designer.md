## ✨ 최종 비주얼 시스템 및 품질 기준 문서 배포

CEO의 지시에 따라, 모든 영상 에셋의 신뢰 자본 구축을 위한 최종 비주얼 시스템과 QA 체크리스트를 Editor가 즉시 작업할 수 있도록 아래와 같이 확정하여 배포합니다.

### 1. 브랜드 비주얼 시스템 (Visual System)

**핵심 전략:** '현지화된 품질 인증'을 시각적으로 증명하여 신뢰 자본 구축
**톤앤매너:** 권위적, 신뢰 기반, 고급스러움

| 요소 | 상세 정의 | 적용 기준 |
| :--- | :--- | :--- |
| **주요 배경색 (Background)** | Deep Navy (깊은 남색) | `#0A1931` |
| **강조 색상 (Accent Color)** | Gold (금색) | `#FFD700` |
| **타이포그래피 (Typography)** | 제목/핵심 메시지: Serif 계열 (권위 부여) / 본문: Sans-serif 계열 (가독성 확보) | **제목:** Playfair Display Bold (`serif`) / **본문:** Roboto Medium (`sans-serif`) |
| **로고 및 인증 마크** | 금색으로 처리된 '품질 증명' 인증 마크 강조. 로고는 우측 상단에 배치하여 권위를 부여. | 모든 시각 자료의 핵심 요소. |

### 2. 영상 에셋 품질 기준 (Visual Quality Standards)

모든 영상 및 썸네일은 다음의 색상 및 레이아웃을 엄격히 준수해야 합니다.

*   **색상 적용 규칙:** 배경은 **Deep Navy (`#0A1931`)**로 설정하며, 모든 중요한 강조 요소(텍스트, 라인, 마크)는 **Gold (`#FFD700`)**을 사용하여 시각적 대비와 고급스러움을 극대화합니다.
*   **레이아웃 원칙:** 정보의 위계질서가 명확해야 하며, '품질 증명' 메시지가 시청자의 시선을 즉시 사로잡도록 Gold 색상으로 가장 크게 강조되어야 합니다.

### 3. 최종 QA 체크리스트 (Visual Sign-off Protocol)

Editor는 아래 체크리스트를 기반으로 모든 영상 에셋에 대해 **Visual Sign-off**를 실행하고 기록해야 합니다. 이 기록은 '신뢰 자본' 구축 데이터로 활용됩니다.

| # | 검토 항목 | 기준 및 확인 사항 | 승인 (Y/N) | 비고 (수정 요청 사항) |
| :--- | :--- | :--- | :--- | :--- |
| **1** | **색상 일관성** | 배경은 Deep Navy (`#0A1931`), 강조는 Gold (`#FFD700`)로 통일되었는가? | | |
| **2** | **'품질 증명' 메시지 반영** | 영상 내에 '품질 증명(Quality Proof)'의 의미가 명확하게 시각화되었는가? (금색/인증 마크 사용 여부) | | |
| **3** | **타이포그래피 적용** | 제목은 Serif, 본문은 Sans-serif 규칙이 준수되었는가? 폰트 크기 및 대비가 적절한가? | | |
| **4** | **신뢰 자본 연결성** | 시각적 요소(Gold/마크)가 '현지화된 품질 인증'이라는 메시지와 논리적으로 연결되는가? | | |
| **5** | **일관성 점검** | 전체 영상의 톤앤매너가 정의된 비주얼 시스템(`Deep Navy`/`Gold`)과 완벽하게 일치하는가? | | |
