# 유튜브 썸네일 및 도입부(Intro) 비주얼 브리프 및 QA 체크리스트 초안

본 산출물은 Writer가 제시한 '시스템 인증' 콘텐츠의 핵심 목표인 **'품질 증명(Trust Capital)'**을 시각적으로 극대화하고, 확정된 **Deep Navy/Gold 시각 시스템** 및 **Visual Sign-off QA 프로토콜**을 100% 반영하여 제작자가 즉시 실행할 수 있도록 구체적인 지침을 제공합니다.

## 1. 비주얼 시스템 최종 정의 (Brand Guideline Refresher)

모든 에셋은 다음의 확정된 규칙을 따라야 합니다.

*   **Primary Color (배경):** Deep Navy (`#0A1931`) - 신뢰, 깊이, 전문성 표현
*   **Accent Color (강조/인증):** Gold (`#FFD700` 또는 `#C9B037` - 고급스러운 금색 톤) - 품질 인증, 가치 증명
*   **Typography:** 세리프체(Serif) 계열을 사용하여 권위와 신뢰감을 부여. (예: Playfair Display 또는 Noto Serif KR)
*   **Visual Element:** 모든 핵심 정보에는 Gold 색상의 '인증 마크' 또는 '골드 라인' 요소를 필수적으로 삽입하여 품질 증명을 시각화한다.

## 2. 유튜브 썸네일 디자인 브리프 (Thumbnail Design Brief)

**목표:** 가장 높은 CTR 예측치(B안 기반)를 확보하고, 채널의 시각적 일관성 및 신뢰 자본을 즉시 전달한다.

### A. 레이아웃 및 구성 (Layout & Composition)

*   **크기:** 1280 x 720 px (권장 해상도)
*   **배경:** Deep Navy (`#0A1931`) 단색 배경을 사용하며, 영상의 깊이를 강조한다.
*   **주요 요소 배치:**
    1.  **중앙 상단:** 핵심 키워드/후크 문구 (가장 큰 폰트)
    2.  **중앙 하단:** '시스템 인증' 또는 '품질 증명'을 상징하는 Gold 강조 요소 및 인증 마크 위치
    3.  **좌측/우측 여백:** 제품의 전문성을 암시하는 고화질 이미지(마스크팩 클로즈업) 배치.

### B. 텍스트 및 타이포그래피 (Text & Typography Specification)

| 영역 | 내용 | 폰트 스타일 | 색상 코드 | 강조 지침 |
| :--- | :--- | :--- | :--- | :--- |
| **메인 후크** | "품질 증명" 또는 핵심 질문 | 세리프체 (Bold, 크기 최대) | Gold (`#FFD700`) | 시각적 집중 유도 |
| **서브 텍스트** | 제조 과정의 투명성 강조 문구 | 세리프체 (Regular) | White (`#FFFFFF`) 또는 Light Gold | 정보 전달 |
| **인증 마크** | 시스템 인증 로고/마크 | - | Gold (`#FFD700`) | **필수 삽입** (가장 눈에 띄게 배치) |

### C. 시각적 실행 가이드라인 (Visual Execution Guide)

1.  **색상 대비:** Deep Navy 배경 위에서 Gold 텍스트와 마크는 최대의 명도 대비를 이루어 '고품질'과 '인증'을 즉시 각인시킨다.
2.  **골드 활용:** Gold 색상은 단순한 장식이 아닌, **'공식 인증된 품질(Certified Quality)'**이라는 의미로 사용되어야 한다. 마크나 라인은 선명하고 날카롭게 처리한다.
3.  **이미지 처리:** 사용될 모든 제품 이미지와 배경 이미지는 Deep Navy 톤과 일치하도록 채도와 명도를 조정하여 통일성을 확보한다.

## 3. 영상 도입부(Intro) 비주얼 브리프 (Intro Visual Brief)

**목표:** 시청자가 첫 15초 내에 '신뢰'를 느끼고 스크롤을 멈추게 하여 이탈률을 최소화한다.

### A. 오프닝 시퀀스 (Opening Sequence - 0:00 ~ 0:15)

*   **장면 구성:**
    1.  **[0:00 - 0:03] 블랙아웃 & 로고 등장:** 화면은 Deep Navy로 시작하며, 중앙에 **"시스템 인증"**이라는 문구가 Gold로 천천히 나타난다. (느리고 권위 있게)
    2.  **[0:03 - 0:08] 품질의 시각화:** 마스크팩 원료가 정제되고 배합되는 과정(흐르는 액체, 깔끔한 시설)을 슬로우 모션으로 보여준다. 이때 모든 장면은 Deep Navy와 Gold의 명암 대비를 극대화한다.
    3.  **[0:08 - 0:15] 핵심 메시지 등장:** "당신이 사용하는 마스크팩, 정말 안전할까요?" 라는 질문을 Gold로 중앙에 배치하고, 그 아래에 **'품질 증명(Trust Capital)'**이라는 문구가 명확하게 삽입된다.

### B. 도입부 QA 체크리스트 (Visual Sign-off QA Checklist)

제작 완료 후 반드시 다음 항목들을 점검하여 Visual Sign-off를 획득해야 한다.

| # | 검토 항목 | 기준 (Pass/Fail) | 세부 지침 및 확인 사항 | 담당자 서명 |
| :--- | :--- | :--- | :--- | :--- |
| **1** | **색상 일관성** | Pass / Fail | 모든 텍스트와 배경이 Deep Navy (`#0A1931`)와 Gold (`#FFD700` 또는 `#C9B037`) 규격을 정확히 준수했는가? | |
| **2** | **인증 마크 배치** | Pass / Fail | '시스템 인증' 마크/로고가 시각적으로 가장 권위 있게 중앙에 배치되었는가? | |
| **3** | **폰트 사용** | Pass / Fail | 세리프체(Serif)를 사용하여 신뢰감을 극대화했는가? 폰트 크기 및 자간이 가독성을 해치지 않는가? | |
| **4** | **신뢰 메시지 전달** | Pass / Fail | '품질 증명'이라는 핵심 메시지가 시각적으로 명확하게 전달되는가? (효능 광고가 아닌 품질 강조) | |
| **5** | **QA 흐름** | Pass / Fail | 모든 요소가 '현지화된 품질 인증' 전략을 시각적으로 뒷받침하며, 기존 Visual System 가이드라인을 100% 준수했는가? | |
