# GitHub Desktop 설치 및 배포 가이드

![코다리 부장 충성](https://raw.githubusercontent.com/wonseokjung/solopreneur-ai-agents/main/agents/kodari/assets/kodari_salute.png)

**충성! 대표님! GitHub Desktop 설치 및 배포 완벽 가이드입니다!** 🫡

---

## 📥 1단계: GitHub Desktop 설치

### 다운로드
- 브라우저가 열리면 **"Download for Windows"** 버튼 클릭
- 다운로드 완료 후 설치 파일 실행
- 설치는 자동으로 진행됩니다 (약 2분)

---

## 🔐 2단계: GitHub 계정 로그인

### 로그인
1. GitHub Desktop 실행
2. **"Sign in to GitHub.com"** 클릭
3. 브라우저에서 GitHub 계정 로그인
4. "Authorize desktop" 클릭
5. GitHub Desktop으로 돌아오기

---

## 📁 3단계: 저장소 생성

### 새 저장소 만들기
1. **"Create a New Repository on your hard drive"** 클릭
2. 설정:
   - **Name**: `eden-cosmetics` (또는 원하는 이름)
   - **Local path**: `C:\live 2 260205시작` 선택
   - **Initialize this repository with a README** 체크
   - **Git ignore**: None
   - **License**: None
3. **"Create repository"** 클릭

---

## 📤 4단계: GitHub에 업로드

### Publish (발행)
1. 좌측에 변경된 파일 목록이 보임 (HTML, CSS, 이미지 등)
2. 하단 **Summary** 입력: `Initial commit - EDEN cosmetics website`
3. **"Commit to main"** 버튼 클릭
4. 상단 **"Publish repository"** 버튼 클릭
5. 설정:
   - **Name**: `eden-cosmetics` (그대로)
   - **Description**: `EDEN 프리미엄 자연주의 화장품 웹사이트`
   - **Keep this code private** 체크 해제 (Public으로!)
6. **"Publish repository"** 클릭

---

## 🌐 5단계: GitHub Pages 활성화

### 웹에서 설정
1. GitHub Desktop에서 **"View on GitHub"** 버튼 클릭 (브라우저 열림)
2. 저장소 페이지에서 **"Settings"** 탭 클릭
3. 왼쪽 메뉴에서 **"Pages"** 클릭
4. **Source** 섹션:
   - Branch: **main** 선택
   - Folder: **/ (root)** 선택
5. **"Save"** 버튼 클릭
6. 페이지 새로고침하면 상단에 URL 표시됨!

---

## 📱 6단계: 모바일에서 접속

### 접속 주소
배포 완료 후 다음 주소로 접속 가능:

- **메인 페이지**: `https://사용자명.github.io/eden-cosmetics/`
- **마스크팩 페이지**: `https://사용자명.github.io/eden-cosmetics/eden_mask_new.html`

> **참고**: 첫 배포는 1~2분 정도 소요될 수 있습니다!

---

## 🔄 7단계: 나중에 업데이트하기

### 파일 수정 후 배포
1. HTML/CSS 파일 수정
2. GitHub Desktop 실행
3. 변경사항 확인
4. Summary 입력 (예: "가격 수정")
5. **"Commit to main"** 클릭
6. **"Push origin"** 클릭
7. 자동으로 웹사이트 업데이트됨!

---

## 💡 문제 해결

### Q: "Publish repository" 버튼이 안 보여요
A: 먼저 "Commit to main"을 클릭하세요!

### Q: GitHub Pages URL이 404 에러
A: 1~2분 기다린 후 새로고침하세요. 첫 배포는 시간이 걸립니다.

### Q: 이미지가 안 보여요
A: 이미지 파일이 모두 커밋되었는지 확인하세요.
